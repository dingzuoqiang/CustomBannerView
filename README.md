# CustomBannerView
一个简单方便的轮播图组件，基于viewpager 基础上进行的封装。
可设置 项目中图片，网络图片， View；
支持循环自动播放，手势滑动切换，item点击事件,可设置 点点的样式宽高、颜色、大小、位置  ，；
可设置蒙层；可设置 是否允许滑动；可设置 是否允许循环。
